How to Redeem Your Token

Got a Code? Here's how to redeem it step-by-step:

1. About the Token

The Suffocate Token is an ERC-20 token on the Polygon network. More info can be found at:
token.watchmesuffocate.com/suffocate.html

2. Set up MetaMask

To receive the token, you'll need a crypto wallet - we recommend MetaMask.

  1. Install the MetaMask browser extension if you haven't already.
  2. Add the Polygon Mainnet manually:
    • Network Name: Polygon Mainnet
    • RPC URL: polygon-mainnet.infura.io
    • Chain ID: 137
    • Currency Symbol: POL
    • Block Explorer URL: https://polygonscan.com

3. Import the Suffocate Token

To view your token in MetaMask:

  1. Open MetaMask.
  2. Click “Import Tokens”.
  3. Paste the token contract address from the official token page:
  4. (Copy from https://token.watchmesuffocate.com/suffocate.html)

4. Redeem Your Code

Go to the Redeem Page and:

  • Enter your redeem code (e.g., WMS-ABCD1234).
  • Connect your wallet using MetaMask.
  • Click “Redeem Now”.
